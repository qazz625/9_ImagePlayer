Project description

This project aims to fulfill everyday image editing, photo filtering demands. The image player has plenty of image editing options from simple cropping, to simple color, grayscale filters, to putting cartoon hats, glasses, etc on a person's face.

The project is python and flask based, using primarily OpenCV and numpy libraries for different kinds of filters. The number and types of filters can also be easily expanded into the future, to make the app even more comprehensive.

The design of the project has been kept simple yet elegant as there are already a number of image editing apps which sometimes overwhelm users with too much clutter and repetitive functionality.
