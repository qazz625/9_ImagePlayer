Contrast

Images having a higher contrast level generally display a greater degree of color.

Hue

Hue represents the color being displayed, as found on a red-green-blue scale, color wheel or grayscale. Part of the HSV colorspace.

Brightness

A measure of intensity after the image has been acquired with a digital camera

Value

Value defines how light or dark a given color or hue can be. Part of the HSV colorspace.

Blur

A blur is a shape or area which you cannot see clearly because it has no distinct outline.

Opacity

Specifies the transparency of an element. Low opacity means you can see through the image.

Crop

To “crop” an image is to remove or adjust the outside edges of an image to improve framing or composition.

Rotate

Image rotation is a common image processing routine with applications in matching, alignment, etc

Pencil Sketch

Sketch of the image if it was made using a pencil using only black colour.

Color Pencil Sketch

Sketch of the image if it was made using a color pencil.

HDR

High Dynamic Range increases the dynamic range. Creates an accurate representation of what you see or an artistic HD feel to the landscapes.

Grayscale

A grayscale image is one in which the only colors are shades of gray. Used in processing datasets for ML.

Emboss

Each pixel of an image is replaced either by a highlight or a shadow, depending on light/dark boundaries.

Median Blur

Blur in which the central element of the image is replaced by the median of all the pixels in the kernel area.

Invert

Invert means flipping the colors to their opposites on the color wheel.

Summer

Filter to make it feel like summer. Can be used to make pictures of landscapes more artistic.

Winter

Filter to make it feel like winter. Can be used to make pictures of landscapes more artistic.

Gradient

The directional change in the intensity or color in an image. A fundamental building blocks in image processing.

Dilation

Dilation adds pixels to the boundaries of objects in an image

AI Crop

Cropping the image using the seam carving algorithm. Crops the image by 20%.

AI Expand

Expanding the image using the seam carving algorithm. Expands the image by 20%.

Dogify

A fun filter which detects a face and applies a dog filter. No change happens if there is no face.

Hatify

A fun filter which detects a face and applies a hat filter. No change happens if there is no face.

Coolify

A fun filter which detects a face and applies a glasses filter. No change happens if there is no face.
